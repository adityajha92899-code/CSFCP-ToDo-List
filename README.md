#  To-Do List

## Description
A simple Python-based command-line application to manage daily tasks. Created for Assignment 04.

## How to Run
1. Install Python.
2. Run `python todo.py` in your terminal.

## Features
- Add new tasks.
- View all current tasks.
- User-friendly menu loop.
